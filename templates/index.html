Entri Bulk Sharing Links

Upload a CSV of domains and get back an XLSX with one Entri sharing link per domain.

1 Entri credentials

Find these in your Entri Dashboard under Application Settings. They're sent securely to this server only — nothing is stored or logged.

2 Sharing config

The JSON config object sent to Entri for every domain. The prefilledDomain field is overwritten automatically with each row from your CSV — leave any value there. See the full reference of available fields.

3 Domains CSV

One domain per row. A header row named domain is optional. Bare URLs work too.
